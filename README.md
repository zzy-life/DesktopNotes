Desktop-Notes
=============

![image](Images/main.png)

它是适用于 Windows 平台的易于使用的便签应用程序。

特点：应用程序极简500kb大小

支持始终置顶，开机启动，主题设置等

怎么更新到最新版？
安装最新版压缩包，解压后，双击Desktop_Notes.msi，选第一个，等待安装即可

**v1.3.0 新增功能：** 样式记忆功能 - 新建便签自动应用上次设置的字体、颜色、透明度
